#### Cálculo de xG (Sabermetrics Simplificado) y Distribución de Poisson

Dado que solo tengo `FTHG` (Full Time Home Goals) y `FTAG` (Full Time Away Goals), utilizaré un enfoque simplificado para el xG. Calcularemos el promedio de goles marcados (AGS) y goles recibidos (AGC) para cada equipo en sus últimos 5 partidos. Luego, para un partido entre Equipo A (local) y Equipo B (visitante), los Expected Goals serán:

*   xG (Equipo A) = (AGS_A + AGC_B) / 2
*   xG (Equipo B) = (AGS_B + AGC_A) / 2

Usaremos estos xG como parámetros de la distribución de Poisson para calcular las probabilidades de diferentes resultados.

**Parma (Serie A):**
*   Últimos 5: 1-4, 0-2, 1-1, 1-1, 0-1
*   Goles marcados: 1+0+1+1+0 = 3
*   Goles recibidos: 4+2+1+1+1 = 9
*   AGS_Parma = 3 / 5 = 0.6
*   AGC_Parma = 9 / 5 = 1.8

**Pisa (Serie A):**
*   Últimos 5: 3-1, 5-0, 0-1, 3-0, 1-2
*   Goles marcados: 3+5+0+3+1 = 12
*   Goles recibidos: 1+0+1+0+2 = 4
*   AGS_Pisa = 12 / 5 = 2.4
*   AGC_Pisa = 4 / 5 = 0.8

#### Pronóstico de Mayor Valor (CLV) y Recomendación de Inversión (Criterio de Kelly)

Para el CLV y el Criterio de Kelly, necesito las cuotas reales del mercado. Dado que no se proporcionan, haré un ejercicio hipotético. El "mayor valor" (CLV) se produce cuando mi probabilidad calculada para un resultado es significativamente más alta que la probabilidad implícita en las cuotas de la casa de apuestas.

**Limitación:** Sin cuotas de casas de apuestas reales para el 25/04/2026, no puedo calcular el CLV o aplicar el Criterio de Kelly de manera directa. Sin embargo, puedo ilustrar cómo se haría, asumiendo cuotas hipotéticas.

**Ejemplo Hipotético para un partido (Parma vs Pisa):**

*   **Mis Probabilidades (aproximadas):**
    *   Victoria Parma: 20% (Cuota justa implícita: 1 / 0.20 = 5.00)
    *   Empate: 25% (Cuota justa implícita: 1 / 0.25 = 4.00)
    *   Victoria Pisa: 55% (Cuota justa implícita: 1 / 0.55 = 1.82)

*   **Cuotas Hipotéticas del Mercado (Ejemplo, con valor para Pisa):**
    *   Victoria Parma: 4.50 (Implica 1 / 4.50 = 22.2%)
    *   Empate: 3.80 (Implica 1 / 3.80 = 26.3%)
    *   Victoria Pisa: 2.10 (Implica 1 / 2.10 = 47.6%)

*   **Análisis de Valor (CLV):**
    *   Para Parma: Mi Prob (20%) < Implícita (22.2%) -> **Sin valor**
    *   Para Empate: Mi Prob (25%) < Implícita (26.3%) -> **Sin valor**
    *   Para Pisa: Mi Prob (55%) > Implícita (47.6%) -> **Valor detectado en la victoria de Pisa.**

*   **Recomendación de Inversión (Criterio de Kelly para Pisa):**
    *   `p` (Probabilidad de ganar para Pisa) = 0.55
    *   `q` (Probabilidad de perder para Pisa) = 1 - 0.55 = 0.45
    *   `b` (Beneficio por unidad apostada, odds - 1) = 2.10 - 1 = 1.10

    *   Fracción de Kelly (`f`) = (b * p - q) / b
    *   `f` = (1.10 * 0.55 - 0.45) / 1.10
    *   `f` = (0.605 - 0.45) / 1.10
    *   `f` = 0.155 / 1.10 = 0.1409

    *   **Recomendación:** Apostar aproximadamente el 14.09% de tu capital a la victoria de Pisa.
